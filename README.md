## 🚀 Featured Projects

### [UAE EV AI Lab](https://github.com/Abaan-Ahmed/uae-ev-llm) — Offline RAG System
> Fully offline LLM pipeline over 136 UAE EV charging stations. Zero cloud API calls.

- **93.3% retrieval accuracy** · **Zero hard hallucinations** across 80 evaluation runs
- React 19 + Vite frontend · Python FastAPI backend · Ollama runtime · NVIDIA RTX 4070
- 6-dimension semantic query parser with Haversine geolocation
- Benchmarked Llama 3, Mistral 7B, Gemma 7B, Phi-3 — Llama 3 won

### [SharePort](https://github.com/Abaan-Ahmed/shareport) — P2P EV Charging Marketplace
> Peer-to-peer EV charging platform. Trustless payments between strangers, enforced on-chain.

- Solidity escrow contract: **deposit → release → withdraw** lifecycle on Sepolia Testnet
- ESP32 hardware controller syncing physical charge sessions to on-chain smart contract state via WebSockets
- React + Node.js + MongoDB + Ethereum full-stack

### [EnergyQuest AI](https://github.com/Abaan-Ahmed/energyquest-ai) — AI Pathfinding Game
- Grid-based adversarial game: human vs AI agent with A\* (admissible heuristic + corrected cost function) and BFS
- Python / Pygame · paired gem mechanics · full technical LaTeX report
